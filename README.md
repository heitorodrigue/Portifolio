## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/heitorodrigue/portfolio.git
```

Entre na pasta:

```bash
cd portfolio
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

A aplicação ficará disponível em:

```
http://localhost:5173
```
